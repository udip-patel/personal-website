# Personal Website - v3

VueJS SPA which uses:

- particles.js
- typed.js
- bootstrap
- font awesome


Page is made up of 4 vue components:
- Hero Unit
- Tools Unit
- Projects Unit
- Footer

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
